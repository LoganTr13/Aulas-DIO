  • Html, Head e Body

    Antes de começar com as tags, deve-se ter entendimento que o Html sendo uma linguagem de marcação, tem "tags iniciais" ao qual se responsabilizam pelo desbloqueio as demais tags.

    • Html- Desbloqueia o uso das demais tags
    • Head- Cabeçalho da página, ao qual colocamos informações não apresentadas ao cliente
    • Body- Corpo da página ao qual colocamos o contéudo a ser apresentado

  • Tags de Texto

    São tags ao quais permitem desenvolver o texto/contéudo do texto.

    1. Tag P

      Permite o desenvolvedor criar paragrafos para seus textos

      Obs:Está tag precisa de sua tag de fechamento

    2. Tag H

      A tag permite criar um titulos ao texto, tendo variações de tamanho, sendo 1 a maior de todas e 6 a menor de todas

      Exemplo h6:

      Este é o tamanho h6

    3. Tag Br

      tag responsavel por criar um espaçamento no documento

    4. Tag Hr

      Cria uma linha no documento, separando-o visualmente

    5. Tag Blockquote

      Usado para campos onde se terá uma citação

      Exemplo:

      Prefiro morrer do que perder a vida

      6. Note: Antigas versões do HTML, possuiam suporte ao comando < font >, ao qual permitia alterações na fonte do documento (Foi substituido pelas funções do CSS).

  • Tags de Formatação (Back)

    Tags utilizadas para formatação do texto

    1. Tag Abbr

      Sua funcionalidade é ajudar com abreviações, sendo usado o parametro title para a resolução da abreviação completa.

      Sua Escrita: < abbr title="Brasil" > BR

      Resultado: BR

      Obs:A abreviação completa aparece se você passar o mouse por cima da abreviação.
    2. Tag Sub

      A formatação serve para subscrever alguma informação, como indicar que o número apresentado é binario e não decimal.

      Exemplo: 10012 = 910

    3. Tag Sup

      A formatação serve para sobrescrever alguma informação, podendo ser utilizado em representação de elevação de um elemento na matemática.

      Exemplo:646 = 68.719.476.736 ou 6,87 * 10-10

    4. Tag Mark

      Sua utilização é para destacar algo no texto, exatamente como um marca texto.

      Exemplo:Você esta Lendo isso?

    5. Tag A

      Esta tag não seria focada em formatação, mas seu linkamento com outras partes, permite uma certa formatação. Ela é utilizada para aplicar links no texto sem mostrar a url.

      Exemplo: Meu GitHub

    6. Tag I

      Permite ativar o Itálico no texto

      Exemplo:Texto Itálico

    7. Tag U

      Permite ativar o Sublinhado no texto

      Exemplo:Texto Sublinhado

    8. Tag Strong

      Permite ativar o Negrito no texto

      Exemplo:Texto com Negrito

  • Tags de Lista (Back)

    Tags ao qual permitem o usuario criar uma tabulação enumerada sendo desordenada (ul) ou ordenada (ol).

    1. Tag UL

      Lista ao qual utiliza simbolos, sem ordem

      Exemplo:

      • item
      • item2

    2. Tag OL

      Lista ao qual utiliza uma enumeração com ordem para listar

      Exemplo:

      1. item
      2. item2

    3. Tag Filha: LI

      Esta tag é filha das duas anteriores, ao qual serve para fazer a linha da lista
